- This is a "slightly" modified version of the official OpenEarable dashboard
- You can use "python server.py" to run it.
- You will need to disable the "developer cache" on your browser, or the page might not be updated as frequently as you want.
- Lines 675 to 706 represents the visual part in the index.html file, which acts as a sudden-movement detector with a sliding bar.
- The metrics to the sudden-movement bar can be found in assets/js/ChartManager.js, lines 241 to 345. You can directly use the pre-defined functions and the logs to obtain sensor values in this file.
